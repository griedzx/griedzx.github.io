我的学习之路

关于本博客

嗨，我是Dai Zixi。这里记录我在学习生物信息领域信息技术过程中的一些想法与收获。

我打算将我的学习内容、面临的挑战和取得的进展记录在这里，以此激励和帮助其他有相同兴趣的人。

本博客同时也会分享一些生活感悟和心得体会。希望能与有共鸣的人交流，共同进步。

我的学习目标

通过在招聘软件上查询相关技术岗位要求，我发现

  1. 熟悉Linux系统
  2. 熟练使用一门编程语言（python优先）
  3. 熟悉r或者shell语言
  4. 有着一定的基因组学等生物学背景知识 往往是最基本的要求，通常还有一些生物信息科研分析中常用的工具软件、数据库的学习要求。在接下来的学习中将逐步推进完善这几个板块。

总的来说：

  • 熟悉各种生物信息分析软件与算法 在导师的项目中实践学习
  • 完善基础的技术要求
  • 尝试搭建属于自己的生物信息项目（要求好高，尝试吧）

正在学习的内容

  • Python 基础与numpy/pandas 库
  • R 语言与生信分析相关包
  • Unix/Linux 命令行操作

我会定期更新本博客记录我的学习进展。

希望这将是一段有意义且有趣的学习之旅！
